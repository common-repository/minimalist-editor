No fuzz post editor - more typewriter, less command prompt.

== Description ==
When you just want to focus on what you're writing, the default Wordpress editor can feel cluttered, with distractions surrounding your content. 

Minimalist Editor turns your post editor into a digital typewriter, with nothing else left on the screen than what you're writing. 

Inspired by editors such as WriteRoom.

Works by installing the plugin and <a href="https://make.wordpress.org/support/user-manual/content/editors/distraction-free-writing/#enabling-distraction-free-writing" target="_blank">activating Distraction Free Writing mode</a>.

= Features =
* Typewriter style post editor
* Light and dark mode
* Sans serif typeface option

== Installation ==
1. Upload the Minimalist Editor plugin folder to the "/wp-content/plugins/" directory, or search for it under Plugins -> Add new in you Wordpress backend. 
2. Activate the plugin through the Plugins menu in WordPress.
3. Activate the Distraction Free Writing Mode in the Wordpress new post page.
4. Find additional options under Appearance -> Minimalist Editor
